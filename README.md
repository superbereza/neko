# Neko 🐈

Спокойный десктоп-котик для macOS: живёт на нижней кромке экрана, спит, гуляет, иногда носится, ест корм и своенравно уходит «за стену» погулять. Нативный Swift, ноль зависимостей.

Спрайт — классический **oneko** (Кэндзи Гото, 1989), 8×4 кадра по 32px.

## Возможности
- **Распорядок дня на потребностях**: энергия / скука / голод + суточный ритм (ночью спит крепче, в сумерки — пик активности, «зумиз»).
- **Кормление** — зажми **⌃⌥⌘X** (работает в любой раскладке): корм сыплется из курсора с паузой, можно водить и насыпать дорожку/горку.
  - Корм — пиксельные катышки с **физикой**: гравитация, масса, трение, отскок. Можно **перетаскивать** мышью и **бросать** по инерции; брошенный из верхней половины экрана — улетает за край и исчезает.
  - Горка **обваливается**: вытащил нижний — верхние осыпаются (держатся только в ямке между двумя).
  - Кот ест **верхние** катышки **по кусочку** (3–4 укуса), сидя за кучкой. Во сне не будится — поест, когда сам проснётся.
- **Перетаскивание кота** мышью (болтает ногами, мягко приземляется на лапы).
- **Уход за стену**: изредка сам идёт к краю, копает и уходит на 3–10 минут, потом возвращается.
- Значок в менюбаре — мордочка кота (из спрайта).

## Сборка и запуск
```bash
./build.sh            # собрать dist/Neko.app
open dist/Neko.app    # запустить
```
Приложение не нотаризовано (ad-hoc подпись). При первом запуске: правый клик по `Neko.app` → «Открыть».

## Управление
- **⌃⌥⌘X** (удерживать) — сыпать корм у курсора.
- Менюбар 🐈 → «Насыпать корм», «Погулять (уйти за стену)», «Выход».
- Перетаскивание мышью — кота и катышки.
